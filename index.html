Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

The most efficient practices to prevent web security attacks are as follows:

  1. Sanitization of data

    This ensures that the data a system accepts is in the right format. This can be done manually or by using third-party solutions like NodeJS.

  2. Secure authorization and authentication

    This practice ensures that authorization and authentication are hard to crack. This can be done manually but is best achieved using a framework such as OneLogin.

  3. CI/CD

    This is an automated process of integrating and delivering code to production. If any changes don't pass the checks, they get rejected and flagged.

  4. Automatic user timeout

    This protects users who have left their credentials open in the browser for a long time.
